This folder is the TON signer keys directory placeholder.

Do not commit or upload real mnemonic, seed_hex, secret_key_hex, or private key JSON files to a public GitHub repository.

For Render env:
TON_SIGNER_KEYS_DIR=/opt/render/project/src/ton-signer-keys

For real payouts, this folder must contain private signer wallet JSON files through a secure private deployment path.
